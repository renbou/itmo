# Запуск  

```python3 process_text.py словарь текст```  

Где __словарь__ это файл в формате:  
> слово кол-во  

А __текст__ это файл, в котором стоит попробовать исправить ошибки по расстоянию Левенштейна (редакторскому расст.), определенного в данном случае такими правилами:
- вставка символа
- удаление символа
- замена одного символа на другой
- разделение строки на две (вставление пробела)  

**_В результате работы_** программы создаётся файл текст_fix.txt, отображающий исходный текстовый файл с примененными исправлениями ошибок, а также информация о том, какие слова не получилось исправить (они не были найдены в словаре, но подходящего слова также не было найдено, см. ниже как определяется подходящее слово), которая выводится в стандартный поток вывода.
